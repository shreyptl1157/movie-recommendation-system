# Movie Recommendation System

A Content-Based Movie Recommendation System built with Python. This project calculates similarity scores between movies using their metadata (like genres) and recommends the top matching films to the user.

## 🚀 Features
* Content-based filtering using text attributes.
* Text vectorization utilizing **TF-IDF (Term Frequency-Inverse Document Frequency)**.
* Similarity checking via **Cosine Similarity**.

## 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** Pandas, Scikit-learn
